# Maker Portfolio Starter

A simple HTML5, CSS3, and JavaScript starter site for a maker portfolio.

## Files

- `index.html` - Main website page
- `styles.css` - Site styling
- `script.js` - Small JavaScript enhancements
- `assets/images/` - Put project photos here
- `assets/videos/` - Put project videos here

## GitHub Pages Setup

1. Create a GitHub repository.
2. Upload these files to the repository.
3. Go to Settings → Pages.
4. Choose the branch you want to publish, usually `main`.
5. Save and wait for GitHub Pages to publish the site.

## Things to Replace

- Your email in the contact section
- Your GitHub URL
- Your LinkedIn URL
- Project names and descriptions
- Image placeholders
- Video placeholder
